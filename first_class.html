Computer

A computer is a electronic device that take input from user and process it to give output. It can store data and perform complex calculations at a very high speed. It is used in various fields like education, business, healthcare, entertainment, etc.

CPU

CPU Stand for Central Processing Unit. It is brain for computer that will process all instruction carefuly

Memory

Memory is a component of a computer that stores data and instructions for processing. It can be categorized into different types, such as RAM (Random Access Memory) and ROM (Read-Only Memory).
